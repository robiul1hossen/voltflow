# Task Management Dashboard

A simple **Task Management Dashboard** built with **Next.js**, **TanStack Query**, and **shadcn/ui** as part of a Frontend Developer technical assessment.

The application demonstrates CRUD operations with a clean UI and responsive layout.

## Features

- Fetch and display tasks from a public API
- Create new tasks using a modal form
- Edit existing tasks
- Delete tasks from the list
- Instant UI updates using TanStack Query mutations
- Responsive design with Tailwind CSS and shadcn/ui

Note: This project uses a mock API, so changes do not persist on the server, but the UI reflects successful operations.
